# Rainfall-Prediction
ML models to predict rainfall in customize locations.

# Project Objective-

This project focuses on evaluating machine learning models for predicting ML models to predict rainfall in customize locations. As a QA/Test Engineer transitioning into AI/ML Engineer, the goal of this project is to apply model testing techniques, performance evaluation, and comparison of algorithms.

Models Implemented and Trained-

1️ . Logistic Regression 2️ . Random Forest

Model Evaluation Strategy -

The following testing metrics were used: Accuracy Confusion Matrix Precision Recall F1 Score Feature Importance
